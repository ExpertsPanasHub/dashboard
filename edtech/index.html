🎯 Твои эксперты: позиция на рынке

🍏 Оксана — Похудение / Нутрициология

🔥 Горячая ниша 📈 Растёт
Рынок ниши: «Здоровье и фитнес» — 25 млрд ₽/год (2-я по выручке на GetCourse)
Топ-20 компаний нарастили выручку на 18% за год. Тренды: нутрициология, превентивная медицина, биохакинг.
Средний чек: 10-50 тыс ₽ (массовый сегмент) + 100+ тыс ₽ (премиум программы)
Прогноз 2026: рост 20-25%, ниша стабильно растёт из года в год благодаря тренду на здоровье

⚡ Алексей — Энерговолна / Эзотерика

🧘 Нишевый спрос 📈 Восходящий тренд
Рынок ниши: «Психология + Эзотерика» — часть сегмента ~45 млрд ₽/год (здоровье + психология)
Спрос на «мягкие» практики растёт на фоне стресса и неопределённости. Астрология, энергопрактики, медитации — тренд у женской аудитории 25-45 лет.
Средний чек: 5-30 тыс ₽ (курсы), 100+ тыс ₽ (наставничество)
Прогноз 2026: рост 15-20%, ниша менее конкурентная, выше маржинальность

🎬 Татьяна — Видео контент

📹 Растущий спрос 📈 Новая волна
Рынок ниши: Видеопродакшн / контент-мейкинг — часть «Профессии» (~30+ млрд ₽/год, #1 ниша на GetCourse)
Рост Reels, TikTok, YouTube Shorts создал массовый спрос на обучение видеоконтенту. Онлайн-курсы по съёмке для бизнеса, блогинг, монтаж — восходящий тренд.
Средний чек: 15-50 тыс ₽ (курсы), 100-300 тыс ₽ (наставничество/продакшн)
Прогноз 2026: рост 25-30%, видеоконтент = must-have для всех бизнесов

🔥 Тренды 2026

📱 Короткий формат — марафоны 3-5 дней вместо длинных курсов
🤖 AI в обучении — курсы по ИИ растут x3, AI-ассистенты в продуктах
👤 Преподаватель-инфлюенсер — личный бренд = главный канал продаж
💰 Рост среднего чека — из-за удорожания трафика школы повышают цены
📉 Конверсия падает — аудитория дольше принимает решения
🏥 Здоровье = #1 растущий сегмент — нутрициология, фитнес, психология

💡 Стратегические рекомендации

🍏 Оксана (похудение): Запуск марафона 990₽ — правильный ход. Рынок 25 млрд ₽, ниша растёт. Далее → upsell на 50-100к программу
⚡ Алексей (энерговолна): Менее конкурентная ниша. Строить воронку через бесплатный вебинар → курс 15-30к → наставничество 150к+
🎬 Татьяна (видео): Самый быстрорастущий спрос. Видеоконтент нужен ВСЕМ экспертам. Позиционирование: «видео для экспертов и бизнеса»
🎯 Цель 70к PLN/мес: Реалистична при 3 экспертах + правильных воронках. Ниши выбраны верно — все в растущих сегментах
